The _Elixir Phrasebook_ is a terse work-in progress website for learning the [Elixir Programming Language](https://elixir-lang.org/). It is based on [Elixir in Action](https://www.manning.com/books/elixir-in-action-third-edition) by Saša Jurić, but uses own examples and provides additional exercises.

This website is maintained by the Swiss software company [Composed GmbH](https://www.composed.ch).

Report issues directly on [GitHub](https://github.com/composed-ch/elixir-phrasebook/issues).
